   IMPORTANT: Import the shared OpenAI singleton from `@/lib/openai` instead of instantiating a new client (per CLAUDE.md mandate: "one new OpenAI() instance"). If `web/src/lib/openai.ts` does not yet exist (Plan 01 may not have run yet in Wave 1), create it first with: `import OpenAI from "openai"; export const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });`
   ```typescript
   import { openai } from "@/lib/openai";
   import { NextRequest, NextResponse } from "next/server";

   export async function POST(req: NextRequest) {
     try {
       const formData = await req.formData();
       const audioBlob = formData.get("audio") as Blob | null;

       if (!audioBlob) {
         return NextResponse.json(
           { error: "Keine Audiodatei empfangen." },
           { status: 400 }
         );
       }

       // Whisper requires File with filename for format detection (Pitfall 3)
       const audioFile = new File([audioBlob], "recording.webm", {
         type: audioBlob.type || "audio/webm",
       });

       const transcription = await openai.audio.transcriptions.create({
         file: audioFile,
         model: "whisper-1",
         language: "de", // Force German for better accuracy
       });

       return NextResponse.json({ text: transcription.text });
     } catch (error) {
       console.error("Transcription error:", error);
       return NextResponse.json(
         { error: "Transkription fehlgeschlagen." },
         { status: 500 }
       );
     }
   }
   ```

**Recording flow:**
1. On click (idle): Create AudioRecorder if not exists, call `recorder.start()`, start elapsed timer interval
2. On click (recording): Call `recorder.stop()`, get RecordingResult blob, clear timer, set state to "processing"
3. On processing: Create FormData with blob, POST to `/api/transcribe`, parse `{ text }` response
4. On transcription success: Call `onTranscription(text)`, set state to "done", after 2s reset to "idle"
5. On error at any stage: Set state to "error", log error, after 3s reset to "idle"

**Timer formatting:** `String(Math.floor(elapsed / 60)).padStart(2, "0") + ":" + String(elapsed % 60).padStart(2, "0")`

**Cleanup:** useEffect cleanup function calls `recorder.destroy()` on unmount.

**Accessibility per UI-SPEC:**
- Button has `aria-label` that updates with state: "Sprachaufnahme starten" / "Aufnahme stoppen" / "Transkription laeuft"
- Processing state: the button text serves as the live announcement
