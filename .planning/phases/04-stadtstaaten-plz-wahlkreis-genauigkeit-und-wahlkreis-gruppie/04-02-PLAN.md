<objective>
Replace the centroid+perturbation+union Stadtstaat path with a true PLZ-area ∩ Wahlkreis-polygon area-share intersection, regenerate the mapping, and lock the result with tests.

Purpose: For Stadtstaat PLZs the current code samples a ~1–3 km cloud around a single Geonames centroid and unions every Wahlkreis polygon it touches, so dense-city PLZs surface 3–4 neighbours (20249 → [18,19,20,21] = 8 MdB instead of the responsible WK 21 only). Computing the real overlap of the PLZ AREA polygon with each Wahlkreis polygon, keeping only Wahlkreise with a meaningful area share (≥ 10%) plus always the dominant one, cuts the noise without ever moving a PLZ to a wrong Wahlkreis or emptying the result.

Output:
- Rewritten Stadtstaat branch in `parse-plz-mapping.ts` using @turf/intersect + @turf/area, with the existing safety filters intact.
- Regenerated `web/data/plz-wahlkreis-mapping.json` (Stadtstaat arrays tighter; non-Stadtstaat entries byte-identical).
- New + updated test fixtures and the regression suite green.
- Updated PLZ-MAPPING-STRATEGY.md describing the new method.
</objective>

       ```
       function resolveStadtstaatByArea(
         plzCode: string,
         expectedWks: Set<number>
       ): Set<number> | null {
         const plzFeat = plzFeatureByCode.get(plzCode);
         if (!plzFeat) return null; // no polygon (Grossempfaenger) -> caller falls back
         const plzArea = turfArea(plzFeat);
         if (!(plzArea > 0)) return null;
         const shares = new Map<number, number>();
         for (const wknr of expectedWks) {
           const wkFeat = wkFeatureByNr.get(wknr);
           if (!wkFeat) continue;
           let isect: Feature<Polygon|MultiPolygon> | null = null;
           try { isect = intersect(featureCollection([plzFeat, wkFeat])) as Feature<Polygon|MultiPolygon> | null; }
           catch { isect = null; } // polyclip can throw on degenerate input -> treat as no overlap
           if (!isect) continue;
           const share = turfArea(isect) / plzArea;
           if (share > 0) shares.set(wknr, share);
         }
         if (shares.size === 0) return null; // no overlap at all -> caller falls back to broad-match
         const dominant = [...shares.entries()].reduce((a, b) => (b[1] > a[1] ? b : a))[0];
         const result = new Set<number>();
         for (const [wknr, share] of shares) {
           if (share >= AREA_SHARE_THRESHOLD || wknr === dominant) result.add(wknr);
         }
         return result;
       }
       ```
