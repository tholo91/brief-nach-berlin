<objective>
Group the wizard's disambiguation results by Wahlkreis and label each card Direkt vs über Liste, so a citizen instantly sees which Wahlkreis each MdB belongs to and which one is directly elected.

Purpose: This is Solution B of the phase. Even after the data fix (plan 04-02) tightens most PLZs to a single Wahlkreis, genuine border PLZs (e.g. 20354 → WK 18 + WK 20) and Berlin districts still legitimately surface two Wahlkreise. Today the cards render as one flat radiogroup, so the user cannot tell which person represents which Wahlkreis. Grouping by Wahlkreis with a clear Direkt/Liste label makes the choice obvious. The selection and submit logic do not change — only the presentation.

Output: A grouped, labelled disambiguation view in `Step3Success.tsx`, designed through a UI skill (NOT hand-written ad hoc).
</objective>
