  <name>Task 2: Make edits publish-safe and revision-backed</name>
  <action>
    Implement edit actions so that:

    - every successful edit writes a new revision snapshot
    - live campaigns overwrite the existing slug in place after moderation
    - failed moderation does not replace the currently active public text
    - creator can recover with another edit instead of requiring admin support

    - public slug updates directly
    - internal history is retained
    - materially different campaigns still belong in a new slug, not in a public version browser
  </action>
