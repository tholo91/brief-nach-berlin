<interfaces>
The existing `generateLetter` produces a system prompt with one fixed Anrede block ("Sehr geehrte/r [Titel] [Name]"). After this plan it accepts:

```typescript
export type LetterLevel = "Bund" | "Land" | "Gemeinde";
export type LetterMode = "primary" | "secondary-stellvertretend";

export interface GenerateLetterOptions {
  // existing fields preserved...
  anliegen: string;
  politician: Politician;
  level: LetterLevel;          // NEW
  mode?: LetterMode;            // NEW, defaults to "primary"
}
```

Anrede mapping:
- Bund → "Sehr geehrte/r Frau/Herr Bundestagsabgeordnete/r [Name]"
- Land → "Sehr geehrte/r Frau/Herr Landtagsabgeordnete/r [Name]"
- Gemeinde → "Sehr geehrte/r Frau/Herr Bürgermeister/in [Name]" (or Stadtrat/Stadträtin if `politician.role` indicates so)

Framing mapping:
- Bund + primary: "strukturelle bundesweite Argumentation, Verweis auf gesetzliche Rahmenbedingungen, ggf. Sondervermögen / Bundesgesetz"
- Land + primary: "landesspezifische Verantwortlichkeit, GG-Bezug wenn passend (z.B. Art. 70 GG: Bildungshoheit der Länder), konkrete Landesinstrumente"
- Gemeinde + primary: "lokale Direktheit, konkrete Verwaltungseinheit, kurzer Weg zur Lösung"
- Bund + secondary-stellvertretend: "Ich schreibe Ihnen, weil ich glaube, dass dieses regionale Problem auch auf Bundesebene Beachtung verdient. [Kontext]. Bitte tragen Sie es in Ihre Fraktion / über die Landesgruppe hinein."
- Land + secondary-stellvertretend: analoge Formulierung mit Landtag-Bezug.
