# Quick 260426-c59 — Polish letter email + kill em dashes + warmer share copy + OG fix Summary

**One-liner:** Eliminated em dashes from letter prompts and email template, rewrote the email in warm founder voice with handwritten Thomas signature and vertical airmail stripe, and fixed stale `vercel.app` domain references in OG metadata and the OG-image badge.

## What was done

### Task 1 — Letter prompt: kill em dashes (commit 3b565ee)
- Added explicit `REGEL`: "Verwende KEINE Gedankenstriche (— oder –). Nutze stattdessen Kommata, Doppelpunkte, Klammern oder kurze Sätze."
- Stripped every em dash from the SYSTEM_PROMPT_TEMPLATE body, the LETTER_WRITING_KNOWLEDGE block, and the TONE_INSTRUCTIONS string for tone level 3. Replacements use commas, colons, or short rephrases.
- Replaced `__MIN_WORDS__–__MAX_WORDS__` (en dash) with `__MIN_WORDS__ bis __MAX_WORDS__` to keep the prompt body fully dash-free.
- The model mirrors its instructions' punctuation, so cleaning the prompt is half the fix; the rule body is the other half.

### Task 2 — Email template overhaul (commit 2b2ec48)
All eight sub-changes (a–h) applied in a single rewrite of `buildEmailHtml.ts`:

- **(a) Vertical airmail stripe:** Wrapped the existing 600px content table in a 2-column outer table. Left column is 8px wide with `bgcolor="#C1121F"` (Outlook/Gmail fallback) plus a `repeating-linear-gradient` matching the existing diagonal stripe palette. Both `<td>` cells use `valign="top"` so the stripe stretches the full height.
- **(b) Warm intro:** Replaced the AI-framed intro with Thomas's exact warmer copy. Kept the existing left-border quote wrapper. No "KI" mention, no em dashes.
- **(c) Subtle copy hint:** Added a small italic 11px fine-print line below the letter block ("Lieber per Hand abschreiben. Aber wenn du den Text zwischendurch in Word brauchst: ins Feld klicken, Cmd/Strg + A, dann Cmd/Strg + C."). Not a button.
- **(d) Caveat web font:** Added `<link href="https://fonts.googleapis.com/css2?family=Caveat:wght@500&display=swap" rel="stylesheet">` to `<head>`. Apple Mail / iOS Mail load this; Outlook/Gmail web fall back via the font-family stack on the signature element.
- **(e) Personal sign-off:** New row between CTAs and footer renders "Vielen Dank, dass du schreibst." (Georgia 14px) followed by "Thomas" in a Caveat / Brush Script MT cursive stack at 32px in the dark blue brand color.
- **(f) Reorder:** Disclaimer block removed from above the CTAs and re-inserted as a smaller (11px), lighter (`#aaaaaa`) block BELOW the footer. New order: CTAs → signature → footer → legal.
- **(g) Em-dash sweep:** Replaced "ausdrucken — das ist immer noch besser" with "ausdrucken. Das ist immer noch besser". Footer "Brief nach Berlin — Deine Stimme zählt." is now "Brief nach Berlin · Deine Stimme zählt." with a middle dot. Dropped "KI-generiert" → "generiert" and folded "KI kann Fehler machen" into the prüfe-Politikerdaten line.
- **(h) SHARE_TEXT import:** Replaced the hardcoded share string with `SHARE_TEXT` from `@/lib/config`, removing duplication. Removed the now-unused `APP_NAME` import.

### Task 3 — Founder-voice SHARE_TEXT (commit 989ff55)

WhatsApp + Twitter share URLs derive from `SHARE_TEXT` and pick up the new copy automatically.
